# WordPress Vulnerable Lab

## 🎯 Objetivo
Realizar pruebas de seguridad en un entorno controlado de WordPress vulnerable.

## 🧱 Entorno
- WordPress local (XAMPP)
- Kali Linux (atacante)
- IP: 192.168.1.10

## 🔍 Reconocimiento
- Escaneo con nmap
- Enumeración con WPScan

## 🚪 Explotación
- Identificación de plugin vulnerable
- Ejecución de ataque

## 🛡️ Mitigación
- Actualización de plugins
- Uso de firewall web (WAF)
